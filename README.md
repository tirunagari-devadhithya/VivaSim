# VivaSim – An Explainable Interview Intelligence System

## Project Overview
VivaSim is an explainable interview evaluation system designed to help students
understand how interview answers are judged and how they can improve.

Unlike existing interview preparation tools that act as black-box systems,
VivaSim focuses on transparency. Every score and feedback generated by the system
is derived from explicit, rule-based evaluation logic.

The system simulates how an interviewer evaluates answers rather than conducting
a conversational interview.

---

## Problem Statement
Many students practice interview questions but receive only generic or unclear
feedback such as “be confident” or “improve clarity.” Most existing tools provide
scores without explaining how those scores are calculated.

This lack of transparency prevents meaningful improvement.

VivaSim addresses this problem by converting subjective interview judgment into
measurable, explainable evaluation criteria.

---

## Key Objectives
- Evaluate interview answers using rule-based NLP techniques
- Provide explainable feedback for every evaluation decision
- Highlight strengths, weaknesses, and improvement areas
- Maintain transparency and reproducibility in scoring
- Track interview readiness through structured evaluation

---

## What VivaSim Is NOT
- Not a chatbot
- Not a conversational AI
- Not based on Large Language Models (LLMs)
- Not a deep learning system
- Not a black-box scoring engine

VivaSim is intentionally rule-based to ensure explainability and academic clarity.

---

## System Workflow
1. A predefined interview question is presented to the user
2. The user submits a textual response
3. The response is preprocessed using NLP techniques
4. Evaluation signals are extracted:
   - Keyword relevance
   - Answer structure
5. A weighted, rule-based score is calculated
6. Explainable feedback is generated
7. The result is stored for traceability

---

## Core Modules

### Question Engine
- Stores predefined interview questions
- Each question includes expected keywords and difficulty level

### NLP Analysis Engine
- Text preprocessing (tokenization, stopword removal, lemmatization)
- Keyword relevance analysis
- Structural analysis of answers

### Explainable Scoring Engine
- Combines evaluation signals using explicit weights
- Produces deterministic and reproducible scores

### Feedback Generator
- Converts evaluation results into human-readable explanations
- Clearly states why scores were reduced or increased

### Response Tracker
- Stores evaluated responses
- Maintains traceability of decisions

---

## Technology Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- NLP: spaCy (rule-based processing)
- Database: SQLite
- Execution: Local (no cloud dependency)

## Installation & Running

### Prerequisites
- Python 3.8+
- pip

### Setup Steps
1. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

2. **Download NLP Model**
   VivaSim uses spaCy's English model. Download it using:
   ```bash
   python -m spacy download en_core_web_sm
   ```

3. **Run the Application**
   ```bash
   python app.py
   ```
   The application will start at `http://127.0.0.1:5000/`.
